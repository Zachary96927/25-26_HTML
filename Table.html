The table tag opens a table. Each row is defined by a tr tag. Each collum within a row is defined by a td tag. The th tag is used for data headers

There are lots of attributes tables can use:

  • Border - Makes a border for the table tag
  • Width - Sets the table or column width
  • Height - Sets the table or column height
